# Exam-SRM-Model
This is an implementation of some models used in exam SRM in Python and also in Excel. Specifically, Weighted Least Squares, Lasso, Ridge, Boosting, and Random Forest. My Python code implements all these models, while my Excel file allows the user to simulate WLS, Lasso and Ridge. The data used is from this [Kaggle dataset](https://www.kaggle.com/datasets/litvinenko630/insurance-claims), which includes both personal information of people insured and also their car information, and whether they have a current claim. Logistic regression is used to predict the chance of an active insurance claim using the other data. Any car information other than the car model was removed as it was highly multicollinear, as the car data is determined by the model. Customer age had a high VIF score, but just barely did not pass the threshold of being multicollinear. In the end, the Random Forrest and Boosting got the highest 10-fold cross-validation scores while the simpler models of Lasso Ridge and WLS had a significantly lower scores. This project was both an intro to Excel for me, and also my first shot at probabilistic programming in Python. It taught me a lot, specifically about how to use some of the top data analytics libraries in Python and how to use Xlookup in Excel. This project was a nice intro for both of those topics and definitely made me more excited for future projects that build off of this knowledge.
